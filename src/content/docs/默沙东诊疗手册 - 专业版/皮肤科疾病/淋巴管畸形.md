# 淋巴管畸形

## (淋巴管瘤; 局限性淋巴管瘤; 囊状水瘤; 洞穴状淋巴管瘤)

淋巴管畸形是由增大的淋巴管构成的隆起的皮损。

皮损通常呈棕黄色，偶尔因血管混合呈淡红色或略带紫色。穿刺皮损可见无色或血性液体。

## 淋巴管畸形的诊断

- 临床评估


由于临床鉴别诊断的范围很广，淋巴畸形的诊断是通过临床和超声或MRI进行的。

## 淋巴管畸形的治疗

- 通常无需治疗


淋巴管畸形通常无需治疗。常见的治疗方法包括硬化疗法和切除术。复发常见。
