# 如何评估精神状态

作者：George Newman, MD, PhD, Albert Einstein Medical Center

已审核/已修订 8月 2023

看法 进行患者培训

精神状态检查用于评估患者的意识水平和意识内容。如果患者积极感知周围的世界，并期待检查者和他们的下一步行动，他们就会被认为是警觉的。如果患者对任何刺激都没有反应，则视为昏迷。

对于所有其他中等意识水平，最好避免仅仅依赖定义不准确的描述性词语（如嗜睡、嗜睡、昏迷），因为这些词语是主观的，无助于其他检查人员评估患者是否正在好转或恶化。此类描述性术语应辅以更详细的基于观察的描述，例如以下内容：

- 是否以及如何唤醒熟睡或看似昏迷的患者

- 患者是否需要重复指导

- 患者的异常是连续的还是间歇的。


如果患者没有醒来，最好记录以下内容：

- 需要什么刺激才能唤醒患者（例如声音、触觉刺激、有害刺激）

- 患者对刺激的反应（例如，非特异性动作、睁眼、言语、合作程度）

- 在恢复到未刺激水平之前，患者在刺激后水平下持续功能的时间


除非患者清醒且警觉，否则无法准确表征意识的内容；尝试这样做通常不值得详细研究，因为结果可能无法反映患者的潜在能力。首先评估患者的注意力广度；一个注意力不集中的患者不能完全配合并妨碍检测。

对于有意识的患者，精神状态检查旨在测试大脑的特定部位。例如，语言和计算问题指向优势半球，空间忽视指向非优势半球，失用症指向对侧大脑半球的对侧感觉运动区域。

任何提示有认知功能障碍的病人都应该进行认知功能各方面的评价，(见边栏 精神状态的检查) 包括对时间、地点、人物的定向能力，注意力，记忆力，语言，计算能力，判断力及逻辑能力。

- 时间、地点和人物的定向

- 注意力

- 记忆

- 口头表达和数学能力

- 判断力

- 推理


只有严重的迟钝、谵妄和 痴呆，才会导致对人的定向力丧失（即不知道自己的名字）;当它作为一个孤立的症状出现时，提示了诈病。

另外，检查时需评价病人对疾病的自知力及与文化教育程度相关的知识储备，同时也需对病人情感和情绪作评价。词汇量通常与受教育程度相关。

如何做精神状态检查



视频

要求患者执行以下操作：

- 让患者完成一个复杂指令，涉及身体的三个部位及进行左右侧的辨别（如：把你右手的大拇指放入你的左耳，然后伸出舌头）。

- 说出简单物件及这些物件的零件名称（如，眼镜和镜头、皮带和皮带扣）

- 让患者进行身体部位的命名，让患者读、书写及重复简单的短语，如果发现异常，则进一步进行其他失语检查。


进行空间辨别觉的检查时，让患者模仿检查者做出简单和复杂的手指姿势及画出钟面、立方体、房屋或五角星；在这一过程中，检测者不仅要评价患者的最终完成情况还要注意观察患者在完成这一动作时的努力程度，从而发现异常的没耐心或过度耐心、写字过小症、偏侧忽略。

让患者使用牙刷、梳子，点燃火柴或弹一个响指从而发现其是否有失用症（完成复杂动作的认知能力）。

### 精神状态的检查

|     |     |
| --- | --- |
| 精神状态的检查指对病人当前的认知功能作出评价，包括评价病人的外貌、行为、异常或古怪的信念和感知（如妄想、幻觉等）、情绪和认知能力（如，注意力、定向能力、记忆力）。<br>任何出现精神状态改变或急、慢性认知功能障碍的患者，都应进行精神状态的检查。对精神状态的评价有很多筛查量表，其中尤为实用的有：<br>- 蒙特利尔认知评估（MOCA）用于一般筛查，因为它涵盖了广泛的认知功能 （例如，注意力、专注力、执行功能、记忆、语言、视觉空间技能、抽象、计算、定向）<br>  <br>- 评估阿尔茨海默病患者时采用简易精神状态检查，因为它侧重于测试记忆力<br>  <br>记录基线结果，每年进行一次检查，每当怀疑精神状态有变化时进行一次复查。<br>患者需被告知精神状态的检查是一项常规检查，不用为此而自感尴尬。<br>检查在一个安静的房间内进行，检查者需确保患者可以清楚地听到提问。如病人不懂英语，则使用其能流利交流的语言。<br>精神状态检查需对认知功能的不同方面进行评价。 检查者必须首先确定患者是否专心，如：让患者立即重复检查者所说的三个词语。对于注意力不集中的病人，进一步检查的意义可能不大。<br>待测认知功能参数及如何测试这些参数的示例包括以下内容： |
| 定向 | 检查患者对人物、地点、时间的定向能力：<br>- 人物：你叫什么名字？<br>  <br>- 时间：今天是几月几日？<br>  <br>- 地点：你现在在什么地方？ |
| 短时记忆 | 让患者在2~5分钟后回忆三件物品。 |
| 长时记忆 | 问病人一个关于过去的问题，比如“你在婚礼上穿了什么颜色的西装？”或者“你的第一辆车是什么牌子的？” |
| 计算 | 使用任何简单的数学测试。序列 7 很常见：要求患者从 100 开始，然后减去 7，然后从 93 中减去 7，等等。或询问患者1.35元等于多少个5分钱。 |
| 找词 | 要求患者在 1 分钟内尽可能多地命名单一类别中的物品，例如衣物或动物物品。 |
| 注意力 | 要求患者向前和向后拼写一个 5 个字母的单词。“世界”是常用的。 |
| 命名 | 拿出一件物品，如钢笔、书或尺等，让患者说出物品的名字和它的一部分。 |
| 执行指令 | 从单独一步的指令开始，如：用你的右手摸摸你的鼻子。然后让患者做一个由三步构成的复杂指令，如：用你的右手拿这张纸，把它对折。把纸放在地上。” |
| 书写 | 请患者写一个句子。句子应该包含一个主语和一个宾语，并且应该有意义。拼写错误可以忽略不计。 |
| 空间定向 | 让患者画出一座房子或一个钟面，并标出一个特定的时间。或让患者画2个交叉的五角星。 |
| 逻辑 | 让患者在 3 或 4 个物体之间确定一个统一的主题（例如，都是水果，都是交通工具，都是乐器）。请患者解释一个中等挑战性的谚语，例如“住在玻璃房子里的人不应该扔石头。” |
| 判断力 | 向患者询问需要良好判断的假设情况，例如“如果你在人行道上发现一封盖章的信，你会怎么做？”放在邮箱里才是正确答案；打开这封信表明有人格障碍。 |
