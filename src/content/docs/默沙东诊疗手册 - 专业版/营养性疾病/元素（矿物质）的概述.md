# 元素（矿物质）的概述

作者：Larry E. Johnson, MD, PhD, University of Arkansas for Medical Sciences

已审核/已修订 7月 2023

看法 进行患者培训

人类需要的六种 **常量元素** 以克来计量，

- 包括四种阳离子：钠，钾，钙和镁，

- 伴随二种 阴离子: 氯和磷，


每天需要量在0.3～2.0g之间。这些宏量元素对于维持骨骼、肌肉、心脏和脑的功能至关重要。

（参见 水钠平衡）

人类需要的 **九种微量元素** 以微克来计量：铬、铜、铁、碘、氟、锰、钼、硒和锌：

- 铬

- 铜

- 氟

- 碘

- 铁

- 锰

- 钼

- 硒

- 锌


膳食指南 对于微量元素的需要量都有明确规定(见表 微量元素的每日摄入量指南)。所有微量元素的摄入量过高都有毒性作用，其中一些（铬）可能是致癌物质。目前尚不清楚是否应将铬视为必需（必需）微量元素（1)。

正常饮食的成年人中不易自然发生微量元素缺乏（碘、铁、锌除外）；婴幼儿由于生长迅速且摄入量差异较大，因而更容易患病。

导致微量元素失衡的原因有：遗传性疾病（如，血色素沉着症、Wilson病）、肾透析、静脉营养、患有 先天性代谢性疾病而需限制膳食的患者，或各种流行的饮食方案。

### 微量元素

| 营养成分 | 主要来源 | 功能 | 缺乏的一些影响和中毒 |
| --- | --- | --- | --- |
| 铬 | 肝脏、加工过的肉类、全谷类食品、调味品 | 促进葡萄糖耐受 | **缺乏：** 可能降低葡萄糖耐受<br>**毒性：** 工作场所接触六价铬 (CrO3) 后的皮肤、肺和胃肠道刺激；可能导致鼻中隔穿孔和肺癌 |
| 铜 | 内脏、贝壳类动物、坚果、干豆、水果干、全谷类食品、豌豆、可可豆、蘑菇、西红柿制品 | 酶组成成分、血细胞生成、骨骼形成 | **缺乏**：营养不良儿童发生贫血，也可出现在Menkes（卷发）综合征中<br>**中毒：** 铜中毒； 可能出现于威尔逊病中 |
| 氟 | 海产品、茶、强化氟的水（氟化钠1.0～2.0ppm） | 骨骼和牙齿的形成 | **缺乏**：易患龋齿、可能发生骨质疏松<br>**中毒**：斑釉、恒牙出现斑影和凹痕、脊柱外生骨疣 |
| 碘 | 海产品、加碘盐、蛋类、奶酪、饮用水（含量不一） | 甲状腺素（T4）和三碘甲腺原氨酸（T3）的合成、胎儿发育 | **缺陷：** 单纯性（胶质性，地方性）甲状腺肿，先天性甲状腺功能减退伴聋哑，胎儿生长和大脑发育受损（克汀病）<br>**中毒**：甲状腺功能亢进或黏液性水肿 |
| 铁 | 许多食物（除了奶制品）－大豆粉、牛肉、肾脏、肝脏、鱼类、家禽、蚕豆、蛤蜊、糖浆、以及谷物和麦片（其生物活性依种类而不同） | 合成血红蛋白和肌红蛋白，细胞色素酶，硫磺铁蛋白 | **缺乏**：贫血、异食癖、舌炎、口角炎<br>**中毒：** 肝硬化，糖尿病，皮肤色素沉着； 可能出现于血色素沉着症中 |
| 锰 | 全谷类食品、菠萝、坚果、茶、豆类、西红柿酱 | 健康骨骼组织；<br>与锰相关的酶成分：糖基转移酶，磷酸烯醇丙酮酸盐羧基激酶，锰超氧化物歧化酶 | **缺乏**： 尚不明了<br>**中毒**：神经症状，类似帕金森病或Wilson病的症状 |
| 钼 | 牛奶、豆类、全谷类食品或面包、深色绿叶蔬菜 | 亚硫酸盐氧化酶、黄嘌呤脱氢酶和醛氧化酶的辅酶成分 | **缺乏**：心动过速、头痛、恶心、反应迟钝（亚硫酸盐中毒） |
| 硒 | 肉类、海鲜、坚果、植物类食品（食物中硒含量因土壤中硒浓度而异） | 谷胱甘肽过氧化物酶和以碘组成的甲状腺激素组成成分 | **缺乏**：Keshan症（病毒性心肌炎）、肌无力<br>**中毒**：脱发、指甲异常、恶心、皮炎、外周神经性疾病 |
| 锌 | 肉类、肝脏、牡蛎及其他海鲜、花生、全谷类食品（植物性的生物利用度不同） | 酶的组成成分、皮肤完整性、伤口愈合、体格生长 | **缺乏**：生长迟缓和性发育迟缓，性腺功能减退，味觉迟钝<br>**中毒**：红细胞小红细胞症、嗜中性细胞球减少症、免疫功能低下 |
