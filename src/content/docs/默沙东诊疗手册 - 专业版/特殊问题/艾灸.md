# 艾灸

作者：Denise Millstine, MD, Mayo Clinic

已审核/已修订 12月 2023

看法 进行患者培训

- [参考文献](#参考文献_v39504762_zh) \|

艾灸（a操作和基于身体的实践）是一种使用传统中药的治疗方法 (1)。 在穴位所在的皮肤上方燃烧干燥的艾叶。草药可以是燃香或线状的形式。

艾灸用于治疗与艾灸相似的病症 针刺，包括疼痛、头痛和慢性疲劳。

艾灸可能造成圆形灼伤（如同香烟灼伤）或水疱。

(参见综合、补充和替代医学概述。)
