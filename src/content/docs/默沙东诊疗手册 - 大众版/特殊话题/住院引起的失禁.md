# 住院引起的失禁

作者：Michael Joseph Pistoria, MEng, DO, Lehigh Valley Hospital - Coordinated Health

已审核/已修订 10月 2023

浏览专业版

住院患者可能会不由自主地漏尿（尿失禁）或排大便（大便失禁）。很多情况下，失禁缘于周围环境，而不是患者的生理功能出现问题。

易于发生失禁的情况如下：

- 严格要求卧床

- 服用利尿剂，使膀胱快速充满尿液

- 由于床太高或病得太严重，不能下床

- 由于手术或疾病等原因使行动困难或疼痛不能下床

- 正在使用一些设备，如静脉（IV）输液、吸氧、心脏监护和导管

- 患有膀胱或肠道感染


在这些情况下，如厕变得非常困难，而且比平时需要更多的时间。

失禁时可选择便盆，尽管有时候不方便使用。需要帮助才能使用便盆或去厕所。痴呆患者、突然神智错乱患者和中风患者都不能够通过按铃呼叫来获得帮助。按铃后，可能回应延迟。这种延迟可能会导致失禁。

有时某些药物或功能紊乱也很容易导致失禁。

（也可参见住院引起的问题。）

## 住院失禁的预防

医护人员可以定时帮助患者如厕。有时在病床旁边放一个便椅（座便器）可方便患者。降低床的高度或重新排列医疗设备会很有用。准备一个尿壶可方便男性患者。确保患者熟悉去厕所的道路，使厕所位置易于识别也有助于防止失禁。
