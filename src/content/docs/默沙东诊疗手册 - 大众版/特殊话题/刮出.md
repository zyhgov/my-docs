# 刮出

作者：Denise Millstine, MD, Mayo Clinic

已审核/已修订 12月 2023

浏览专业版

刮痧（一种手法和基于身体的实践），指在皮肤上（通常在背部、颈部或四肢）使用钝性工具摩擦。刮痧在田径运动中很流行，尤其是举重。据认为，刮痧可增加流向某个区域的血液并增强新陈代谢和愈合。根据所用工具，刮痧也可以称为硬币刮痧或 **勺子刮痧**。

这些疗法对绝经期症状、肌肉骨骼疾病和疼痛表现出益处。

刮痧通常会导致皮肤淤青。
