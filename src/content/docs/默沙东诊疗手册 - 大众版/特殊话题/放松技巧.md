# 放松技巧

作者：Denise Millstine, MD, Mayo Clinic

已审核/已修订 12月 2023

浏览专业版

放松是一种心身疗法，包括专门为缓解紧张和压力而设计的医学实践。特定技术可能针对以下各方面：

- 控制应激反应（通过交感神经系统）

- 降低血压和心率

- 减轻肌肉紧张

- 改变脑电波活动


放松技术可包括冥想、引导意象或催眠疗法或将这些方法结合。放松疗法可以帮助缓解抑郁，并可用于缓解焦虑和某些类型的疼痛。
