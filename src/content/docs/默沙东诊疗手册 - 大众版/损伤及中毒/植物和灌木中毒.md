- 通常不太可能中毒，除非植物毒性大或摄入量大（例如，如果把叶子或其他部分浓缩到酱中或焙成茶）。

- 具有强毒性而且潜在可致命的植物包括蓖麻和相思豆、毒芹和水毒芹以及夹竹桃和毛地黄。

- 只有少数植物毒物具有特异性解毒剂，因此医生通常仅治疗患者的症状。

许多其他植物产生的毒性作用不太严重（参见表中等毒性植物）。

## 蓖麻和相思豆

蓖麻中含有的蓖麻毒素是一种极度强效的毒素。蓖麻毒素曾被用于暗杀，它是一种大规模杀伤性武器。蓖麻籽的壳非常硬，因此蓖麻必须咀嚼方可释放毒素。

相思豆中含有的相思豆毒素是一种毒性甚至比蓖麻毒素更强的相关毒素。如果吞下，可能会致命。儿童仅需咀嚼一颗豆即可死亡。

蓖麻或相思豆中毒可延迟一段时间后导致严重的呕吐和腹泻（通常为血性）。中毒者晚期出现神智错乱和癫痫发作。进而可能昏迷甚至死亡。

有时医生会尝试在毒物被身体吸收之前将这些豆子清除出胃和小肠。目前尚无解毒剂，但医生会通过静脉（静脉内）和其他治疗方式为患者补液，以治疗他们的症状。

## 毒芹

毒芹中毒可在15分钟内导致症状出现。中毒者感到口干并随后出现心跳加快、震颤、出汗、癫痫发作以及肌无力。水毒芹可导致呕吐和腹泻、谵妄、癫痫发作及昏迷。目前尚无解毒剂，但医生会为患者提供治疗症状的护理。

## 夹竹桃、毛地黄和铃兰

夹竹桃、毛地黄和铃兰含有一种与心脏药物地高辛非常相似的物质。这些植物可引起呕吐和腹泻、意识模糊、心跳不规则和高血钾水平（高钾血症）。医生有时会用 DIGIFab 治疗患者，DIGIFab 是一种用于治疗地高辛过量的药物。

### 一些中等毒性植物

| 植物 | 症状 | 治疗 |
| --- | --- | --- |
| 芦荟及相关植物 | 胃肠炎和肾脏炎症 | 如果吞食植物则进行支持治疗\*，若皮肤受到植物刺激则用肥皂和水进行冲洗 |
| 杏子、野樱桃和桃核及苹果和其他种子（ _李属_ 和 _苹果属_ 种属），它们含有一种在进食后转化为氰化物的化学物质——通常仅当咀嚼和吞咽许多种子时 | 出现缺氧的症状，比如恶心、头晕、注意力难以集中、头痛、呕吐、嗜睡和协调性差（与氰化物中毒的症状类似） | 重度中毒时，可静脉给予羟钴胺素并使用氰化物解毒包（内含硝酸戊酯吸入剂和供静脉输注的亚硝酸钠和硫代硫酸钠） |
| 马兜铃 | 肾脏中形成瘢痕组织 | 支持疗法\* |
| 杜鹃花 | 胆碱能†症状 | 支持疗法\*和阿托品 |
| 含草酸钙的植物，包括水芋叶、花叶万年青（花叶万年青属）、象耳蕨、三叶天南星、喜林芋属、大黄和许多其他相关植物 | 口腔因叶子中草酸钙晶体受到刺激 | 支持疗法\*和用冷液体冲洗口腔以舒缓该区域并清洗刺激物 |
| 辣椒及相关植物（辣椒） | 对皮肤和黏膜的刺激 | 支持疗法\*和使用牛奶或冰淇淋来溶解刺激物 |
| 含秋水仙碱的植物（秋水仙、藏红花或嘉兰） | 延迟的胃肠炎和多个器官系统的障碍（衰竭）<br>妨碍骨髓产生血细胞和血小板的能力，可能导致贫血，感染和/或出血 | 支持疗法\* |
| 颠茄 | 抗胆碱‡症状、高热、癫痫和幻觉 | 支持疗法\*<br>导致极度高热或癫痫的可能是毒扁豆碱 |
| 蚕豆 | 体内缺乏 G6PD 酶（可保护红细胞）的人会出现胃肠炎、发热、头痛和溶血性贫血 | 支持疗法\*<br>对于严重贫血和中毒，考虑逐渐将血液移除并替换为新鲜的供体血（换血疗法） |
| 绿色土豆和土豆芽 | 胃肠炎、幻觉和谵妄 | 支持疗法\* |
| 冬青浆果 | 胃肠炎 | 支持疗法\* |
| 蔓陀罗 | 抗胆碱‡症状、高热、癫痫和幻觉 | 支持疗法\*<br>导致极度高热或癫痫的可能是毒扁豆碱 |
| 甘草（生植物，而不是常见糖果\[现已不再用甘草植物提取物调味\]） | 血钾过低、高血压和液体积聚导致的肿胀（水肿） | 支持疗法\* |
| 铃兰、毛地黄和夹竹桃——均含有与心脏病药物地高辛类似的物质 | 血钾过高和不正常心心律（心律失常） | 支持疗法\*和抗洋地黄抗体 |
| 槲寄生 | 胃肠炎 | 支持疗法\* |
| 乌头 | 心率缓慢、心律失常、麻木和刺痛，以及虚弱 | 支持疗法\*<br>有时静脉输入碳酸氢钠 |
| 荨麻 | 皮肤刺痛和烧灼感 | 支持疗法\* |
| 龙葵，常见的或木本的 | 胃肠炎、幻觉和谵妄 | 支持疗法\* |
| 龙葵，致命的 | 抗胆碱‡症状、高热、癫痫和幻觉 | 支持疗法\*<br>导致极度高热或癫痫的可能是毒扁豆碱 |
| 除蚤薄荷 | 肝损害（严重时导致黄疸、意识混乱和出血倾向） | 乙酰半胱氨酸 |
| 喜林芋和相关植物 | 因其叶中的草酸钙晶体导致口腔损害 | 支持疗法\*和用冷液体冲洗口腔以舒缓该区域并清洗刺激物 |
| 一品红 | 如果接触到口腔黏膜、鼻道、阴道或尿道会出现轻度刺激 | 无需治疗 |
| 毒漆藤 | 严重皮肤瘙痒和粘膜刺激 | 冷敷、抗组胺药，有时使用皮质类固醇 |
| 美洲商路 | 胃肠炎<br>如果接触到口腔黏膜、鼻道、阴道或尿道会出现刺激 | 支持疗法\* |
| 石柑属植物 | 因其叶中的草酸钙晶体导致口腔损害 | 支持疗法\*和使用牛奶或冰淇淋来溶解晶体 |
| 紫杉 | 胃肠炎<br>极少情况下出现癫痫、心律失常甚至昏迷 | 支持疗法\* |
| \*支持治疗包括静脉输液（静脉注射）、维持机体功能的治疗（比如退热药）、血压下降时的升压药以及呼吸机。 |
| †胆碱能症状包括心率缓慢、心脏收缩减弱、危险的低血压、呼吸困难（因气道收缩所致）、面色潮红、腹部绞痛、腹泻、排尿和唾液分泌增多、流眼泪、出汗增加和肌肉抽搐。 |
| ‡抗胆碱能症状包括意识混乱、视物模糊、便秘、口干、头晕目眩、排尿困难，甚至是尿失禁。 |
| G6PD = 葡萄糖-6-磷酸脱氢酶。 |
