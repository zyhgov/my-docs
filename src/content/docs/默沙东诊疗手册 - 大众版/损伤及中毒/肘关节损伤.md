# 肘关节损伤

作者：Paul L. Liebert, MD, Tomah Memorial Hospital, Tomah, WI

已审核/已修订 11月 2023

浏览专业版

损伤可发生于附着于肘关节的肌腱。肱骨外上髁炎（网球肘）在将手部向后牵拉的肌腱发炎时出现。肱骨内上髁炎（高尔夫球肘）是将手掌向腕部弯曲的肌腱发炎时出现。对于肘部骨折，参见肘关节骨折。

（另见运动损伤概述。）

### 肘关节受伤

|     |
| --- |
| 网球肘及高尔夫球肘引起肘关节和前臂不同位置的疼痛。<br> |
