# 初级保健医生

作者：Michael R. Wasserman, MD, California Association of Long Term Care Medicine

已审核/已修订 4月 2023

浏览专业版

- [了解更多信息](#了解更多信息_v36200467_zh) \|

通常，初级保健从业者（也称为初级保健医生 \[PCP\]、全科医生或家庭医生）应成为医疗保健系统的切入点，初级保健从业者通常是医生，但有时也可能是护士或内科医生的助手。初级保健医生提供一般的医疗服务，并负责整体护理，包括与医学专家的协调及支持性护理。某些健康保险计划要求被保险人选择初级保健医生。但是，即使患者的医疗保险不要求，初级保健医生也有许多优势，可以带来更好的护理。（也可参考 充分利用医疗保健的简介。）

有初级保健医生的人很少会不恰当的去急诊室，也很少被不认识的医生接诊。当人们不认识就诊医生时，医生没有有助于诊断和治疗疾病的病史信息。结果就是，医生可能需要重复检查或做不必要的检查。

如果人们与初级保健医生建立了稳定的联系，则可以更顺畅地进行沟通，也更方便做出医疗决策。人们更倾向于信任他们认识的医生，而且当出现医疗问题时，更不容易焦虑。初级保健医生通常与患者保持长期的关系，并建立了双方都接受的沟通渠道。他们熟悉患者的需求和价值观、患者能够接受信息的最佳方式、患者如何应付逆境、患者按处方买药的能力和依靠的家庭成员等。

初级保健医生会向人们解释需要哪种保健，为什么需要，以及多久进行一次健康咨询。他们提供适当的预防性医疗保健服务，包括疫苗接种，在需要时将患者转诊给专科医生，并与其他医疗保健从业人员协调护理事宜。一些卫生保健计划要求人们在就诊专家前向他们的初级保健医生咨询。虽然这种做法可视为专业护理的障碍，但是它是有益的。例如，转诊应确保初级保健医生了解个人所有的卫生保健需求。它还确保人们去看合适的专科医生，并且专科医生会得到患者适当的背景信息。

确定初级保健医生前，人们可向朋友或亲戚寻求建议。或者打电话给医学院或当地医院，咨询某个科室，例如儿科、内科或家庭医疗。患者有关个体医生的在线调查数据是另一种信息来源，但不应是唯一的来源，也不应替代个人评估。许多健康保险计划对医生和其他保健人员的选择有限制。这时，人们应当了解计划内容并得到可选的保健人员名单。有时人们不能见到他们选择的医生，因为这些机构不再接受新患者。

在选择初级保健医生时，人们应考虑什么对于自己最重要（例如友好、非常仔细、耐心或及时）。有些人希望医生花更多时间与其接触，即便这可能使得预约的时间很晚。还有些人更喜欢医生能准时赴约，即使这样做可能会使看诊时间受限。老人可能希望找到擅长老年人群治疗的初级保健医生（老年科医生）。人们应当寻找他们相处融洽并信任的医生。对于某些人而言，这意味着选择一名能够得心应手地使用远程医疗的医生。

关于医生的资格信息，人们可以给美国医疗专家委员会打电话（866-275-2267 免费）或者检索该机构的网站（ [www.abms.org](http://www.abms.org)）。

### 这位医生是合适的人选吗？

|     |
| --- |
| 向医生提问如下问题，有助于你了解他是否合适：<br>- 医生是否参与于我的健康保险计划？<br>  <br>- 医生的正常办公时间是几点？<br>  <br>- 例行就诊的通常等候时间？ 是否可以紧急就诊？<br>  <br>- 医生是否接受电话或邮件咨询（办公时间和下班时间）？ 如果接受，他的回复速度如何？<br>  <br>- 医生是否有在线门户网站，用于与患者沟通（如处方更新请求和查看检查结果）和时间安排？<br>  <br>- 医生是否提供方便的远程医疗服务？ <br>  <br>- 医生的电子记录系统是否与我可能会被转过去的医院和其他医生整合在一起？ <br>  <br>- 如果我需要住院，医生会照料我吗？还是需要将我转给其他医生？ 他在哪些医院任职？<br>  <br>- 他有资质认证吗？<br>  <br>- 他的办公室进出容易吗？<br>  <br>- 他守时吗？<br>  <br>- 非办公时间(夜间或周末)或他不在时，谁负责他的患者？ 当其他人负责时，是否知道有办法获得患者的健康信息？<br>  <br>- 还有哪些人通常参与这位医生的患者的保健？ 例如，护士、执业护士或医生助理会在多大程度上给我提供护理？<br>  <br>- 检查结果(正常的和异常的)怎样沟通，谁(医生还是患者)主动提出沟通？<br>  <br>询问这位医生的患者的几个的有用问题：<br>- 他愿意花时间倾听患者的顾虑吗？<br>  <br>- 他会详细解释疾病诊断吗？<br>  <br>- 你相信他的观点吗？<br>  <br>- 在开具药物处方之前，医生是否讨论其益处与风险？<br>  <br>- 在开具药物处方之前，医生是否讨论其替代药物？ |
