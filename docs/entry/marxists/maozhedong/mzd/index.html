毛泽东选集​

第一卷​

Ø 第一次国内革命战争时期
内容概要：收录毛泽东在1925-1927年间的著作，包括《中国社会各阶级的分析》《湖南农民运动考察报告》等，奠定中国革命理论基础。

第二卷​

Ø 第二次国内革命战争时期
内容概要：涵盖1927-1937年土地革命战争时期的文献，如《星星之火，可以燎原》《反对本本主义》，提出农村包围城市战略。

第三卷​

Ø 抗日战争时期（上）
内容概要：1937-1941年间的著作，包括《实践论》《矛盾论》《论持久战》，系统阐述辩证唯物主义与抗日战略。

第四卷​

Ø 抗日战争时期（下）
内容概要：1941-1945年文献，如《新民主主义论》《整顿党的作风》，构建新民主主义理论体系。

第五卷​

Ø 第三次国内革命战争时期
内容概要：1945-1949年解放战争时期的文稿，包括《论联合政府》《将革命进行到底》，指导中国革命胜利。

第六卷​

Ø 社会主义革命和建设时期（一）
内容概要：1949-1956年间的文献，如《论十大关系》《关于正确处理人民内部矛盾的问题》，探索社会主义建设道路。

其他​

Ø 未编入选集的重要文献
内容概要：包括书信、批示、讲话稿等，补充展现毛泽东的多维思想。

PDF文库​

Ø 毛泽东著作全集（数字化版）
说明：提供毛泽东著作的PDF版本，支持全文检索与下载。

参考资源​

权威版本​

Ø 《毛泽东集》（十卷本）及《补卷》（九卷+年表）
特点：日本学者竹内实主编，收录最全面的毛泽东著作及年谱。

历史文献​

Ø 1968年武汉版《毛泽东思想万岁》
背景：特殊历史时期编纂的毛泽东言论汇编，反映时代特色。

文库使用说明​

  1. 访问方式

    • 在线阅读：直接点击链接跳转至马克思主义文库（Marxists.org）中文页面。
    • 离线下载：通过PDF文库批量获取文献，支持学术研究与教学使用。

  2. 注意事项

    • 部分文献因历史原因存在版本差异，建议对照权威纸质版（如人民出版社《毛泽东选集》）校核。
    • 引用时需注明原始来源及网络地址。

结语
毛泽东文库是研究中国革命史、思想史的核心资源库，其文献不仅是历史的见证，更是理解中国现代化进程的钥匙。通过系统整理与开放共享，这些资源将持续为学术研究与思想传承提供动力。
